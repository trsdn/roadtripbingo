üìù Technical Solution

Problem: CSS was using fixed dimensions that forced all images into squares.

Solution: Changed CSS to use responsive sizing that preserves aspect ratios:

/* Before (broken) */
.icon-item img {
    width: 50px;
    height: 50px;
    object-fit: contain;
}

/* After (fixed) */
.icon-item img {
    max-width: 50px;
    max-height: 50px;
    width: auto;
    height: auto;
    object-fit: contain;
}

Result: Non-square icons (like DD.png and WOB.png at 520x110 pixels) now display correctly without distortion.
